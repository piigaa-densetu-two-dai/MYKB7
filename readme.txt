カシオ製MSX・PV-7用基本スロットエキスパンダー ～俺なりのKB-7～

御注意：回路図は手書きメモを元に書いたものなので間違いがある可能性があります。

PV-7用の基本スロット拡張回路です。純正拡張ユニットKB-7の代用品です。
PV-7に限らずSLOT2,SLOT3が無い機種全般に応用できると思います。

使用部品

* XilinxのXC9536/XC9572 PLCC44タイプ
末尾にXLが付いたバージョンは使えません。(要電源回路)
取り外し品がaliexpressにたくさんあります。150～200円位でしょうか。

* 上記のソケット
aliexpress、秋月電子等にあります。

* 2.54mmピッチのカードエッジコネクタ(50P)
aliexpressにあります。5個で300～400円位でしょうか。

* 抵抗、コンデンサ（パスコン）
220ohm、0.1uF

* 蛇の目基板
秋月電子のＢタイプというサイズのものが丁度良いです。

作り方

XilinxのISE14.7でSLOT.vhdをビルドして XC9536かXC9572に書込みします。
XC9536用にビルド済のSLOT.jed、SLOT.svfも付けておきました。

あとは回路図通りに配線すれば完成です。
